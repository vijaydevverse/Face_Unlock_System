# 🔓 Face Unlock System

A Python-based Face Recognition Authentication System that uses Computer Vision and AI to register and verify a user's face through a webcam. The system grants access only when the detected face matches the registered face image.

## 🚀 Features

* Face Registration using Webcam
* Real-Time Face Recognition
* Secure Face Matching
* Access Granted / Denied Authentication
* OpenCV Integration
* Face Recognition Library Integration
* Easy Setup and Lightweight Project

## 🛠️ Technologies Used

* Python
* OpenCV
* Face Recognition
* NumPy

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-link>
cd Face_Unlock_System
```

Create virtual environment:

```bash
python -m venv venv
```

Activate environment:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Usage

### Register Face

```bash
python face_set.py
```

Press **S** to save your face.

### Unlock System

```bash
python face_unlock.py
```

Show your face to the webcam for authentication.
